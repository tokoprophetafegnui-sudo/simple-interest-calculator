# Simple Interest Calculator

A simple yet powerful command-line tool to calculate simple interest based on user input.

## Project Description

The Simple Interest Calculator is a Bash script application designed to compute simple interest for financial calculations. Users can input the principal amount, rate of interest, and time period to quickly determine the simple interest earned or paid.

## Features

- Easy-to-use command-line interface
- Fast and accurate simple interest calculations
- Flexible input for principal, rate, and time period
- Lightweight and portable Bash script

## Formula

Simple Interest = (Principal × Rate × Time) / 100

## Usage

```bash
bash simple-interest.sh
```

Follow the prompts to enter:
- Principal amount
- Rate of interest (per annum)
- Time period (in years)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/tokoprophetafegnui-sudo/simple-interest-calculator.git
   ```

2. Navigate to the directory:
   ```bash
   cd simple-interest-calculator
   ```

3. Run the script:
   ```bash
   bash simple-interest.sh
   ```
